# Superstore-Interactive-Dashboard
This repository contains an interactive Tableau dashboard built to analyze sales, profit, and order quantity metrics from the Sample Superstore dataset. The dashboard provides fully dynamic filtering, enabling users to explore performance by state, category, segment, ship mode, and sub-category.

- Allow interactive selection for deep-dive insights
